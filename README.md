# Neptune

Neptune is a self-hosted media server that runs your own movie and show library as live TV
channels with a real guide, plus on demand. One program: library, metadata, channels, guide,
subtitles, downloads, and clients, connected on the inside. The server runs on your machine;
the clients connect only to it. No account, no subscription, no telemetry.

This repository hosts Neptune's downloads, release notes, and issue tracker. The source is not
public. Grab the latest build from the Releases page - every release carries the same stable
file names, so links never go stale. Found a problem? Open an issue here.
